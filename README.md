# Dashboard проект

Интерактивный дашборд для анализа данных по компаниям и проектам, построенный на React + TypeScript + Vite.

## Описание

Проект представляет собой современный веб-интерфейс для визуализации и анализа ключевых показателей эффективности (КПЭ) по компаниям и проектам. Включает в себя:

- **Сводные карточки** с основными метриками (численность на объекте, дефицит, потребность, лимиты)
- **Интерактивные графики** распределения персонала по компаниям и проектам
- **Таблицы анализа** план-факт экономической эффективности с сортировкой
- **Адаптивный дизайн** с использованием современных UI-компонентов

## Технологический стек

- **React 19** - библиотека для построения пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript для надежности кода
- **Vite** - быстрый инструмент сборки и dev-сервер
- **Recharts** - библиотека для построения графиков
- **Lucide React** - набор иконок
- **Day.js** - работа с датами
- **Zod** - валидация данных

## Установка и запуск

```bash
# Установка зависимостей
bun install

# Запуск dev-сервера
bun run dev

# Сборка для production
bun run build

# Предварительный просмотр production-сборки
bun run preview
```

## Тестирование

```bash
# Запуск UI-тестов с Playwright
bun run test:ui
```

## Структура проекта

```
src/
├── components/dashboard/   # Компоненты дашборда
├── data/                   # Mock-данные для разработки
├── lib/                    # Утилиты (форматирование и т.д.)
├── services/               # Адаптеры данных
└── types/                  # TypeScript типы
```

## Разработка

Проект использует ESLint для проверки качества кода. Конфигурация находится в `eslint.config.js`.
